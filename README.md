# Portfolio & Gungnir Analytics

Este repositorio contiene mi sitio personal y la plataforma **Gungnir**, una herramienta de analítica para el ecosistema de innovación.

## Estructura
- `index.html`: Landing page y portafolio personal.
- `Gungnir.html`: Prototipo interactivo de la plataforma Gungnir.
- `tokens.css`: Sistema de diseño compartido (basado en el stack de diseño de Gungnir).

## Stack Técnico
- **Frontend:** HTML5, CSS Custom Properties (Tokens), React (vía CDN para el prototipo).
- **Backend:** Python (Flask/Gunicorn) - *Próximamente para el motor de IA*.
- **Deployment:** Vercel (Frontend) y Railway (Backend).
